Customer Segmentation Using RFM Analysis and K-Means Clustering

Business Problem:
Companies need to understand customer purchasing behavior to improve retention and marketing strategies. 
This project uses transaction data from the UC Irvine Machine Learning Repository to identify customer segments based on purchasing behavior.

Methodology:
1. Cleaned retail transaction data
2. Created revenue metrics
3. Calculated RFM features:
   - Recency
   - Frequency
   - Monetary
4. Applied log transformation
5. Scaled features
6. Used K-Means clustering
7. Developed customer profiles and strategies

Results:
Segment	Description
VIP Customers -	High-value frequent buyers
New & Growing Customers -	Recent customers with growth potential
At-Risk Valuable Customers - Previously valuable but inactive
Inactive Customers -	Low engagement customers
